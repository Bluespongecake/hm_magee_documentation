# Documentation Layout

- `docs/reference/`: source/reference HTML exports used for visual/content comparison.
- `docs/plans/`: planning/prototype page outputs.
- `docs/tokens/source/`: non-runtime token source artifacts (for example AI/export JSON).
- `docs/tokens/guides/`: token mapping and usage documentation.

Runtime app files remain at repo root (`app/`, `public/`, `design-tokens.css`, Next.js config files).
