# Component Token Map

Reference guide only. Runtime tokens for implementation live in `design-tokens.css` at repo root.

## Header
- Height: `--component-header-height`
- Background: `--component-header-background`
- Border: `--component-header-border-color`
- Nav text: `--component-header-nav-text`
- Active nav text: `--component-header-nav-text-active`
- Active nav indicator: `--component-header-nav-active-indicator`

## Side Navigation
- Width: `--component-side-nav-width`
- Background: `--component-side-nav-background`
- Border: `--component-side-nav-border`
- Link text: `--component-side-nav-link-text`
- Link hover background: `--component-side-nav-link-hover-bg`
- Link active background: `--component-side-nav-link-active-bg`
- Link active indicator: `--component-side-nav-link-active-indicator`

## Button
- Primary background: `--component-button-primary-bg`
- Primary text: `--component-button-primary-text`
- Destructive background: `--component-button-destructive-bg`

## Input
- Border default: `--component-input-border-default`
- Border error: `--component-input-border-error`
- Padding inline: `--component-input-padding-inline`
- Padding block: `--component-input-padding-block`

## Callout
- Info border: `--component-callout-info-border`
- Success border: `--component-callout-success-border`
- Warning border: `--component-callout-warning-border`
- Error border: `--component-callout-error-border`
